## Reflection
Creating this presentation helped me understand that digital literacy is not just about using technology — it is about using it responsibly, critically, and ethically. Using Canva's drag-and-drop interface, I was able to organize ideas visually and communicate complex concepts in a simple, accessible format suitable for first-year students.
